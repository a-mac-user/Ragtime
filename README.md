# Ragtime   拉格泰姆
用python写的CMDB系统
实现了如下功能：
•   存储各种IT资产信息
•   数据可手动添加
•   硬件信息可自动收集
•   硬件信息可自动变更
•   可对其它系统灵活开放API
•   API接口安全认证
